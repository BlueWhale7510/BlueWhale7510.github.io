### 基于混合检索和结果重排的高级 RAG 系统（项目负责人）
**时间：2025.10**
- 针对复杂或表述不佳的用户问题，构建可通过查询重写（Query Rewriting）和结果重排（Re-ranking）优化检索质量的高级 RAG 系统。
- 下载 Llama 2 官方论文作为知识库源文件，调用 DeepSeek API 生成 4 个同意图的相似问题。
- 同时构建稀疏检索索引（BM25）与稠密检索索引（向量数据库），并基于多问题分别检索。
- 使用 RRF 实现结果融合，形成候选文档列表。
- 使用 Cross-Encoder 对候选文档进行重排，得到 Top-5 文档片段并驱动 LLM 生成最终答案。
